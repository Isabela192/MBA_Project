SOLID Bank API

Uma aplicação financeira desenvolvida com princípios SOLID e padrões de projeto

Padrões de Criação

Factory Method para criação de usuários e contas bancárias, garantindo a extensibilidade do sistema.

Padrões Estruturais

Proxy Pattern para controle de acesso e logging nas operações de contas bancárias.

Padrões Comportamentais

Command Pattern para operações financeiras como depósitos, saques e transferências.

Sobre o Projeto

Este projeto demonstra a aplicação de princípios SOLID e padrões de projeto no desenvolvimento de uma aplicação financeira. A arquitetura foi projetada para maximizar a manutenibilidade, testabilidade e extensibilidade do código.

Endpoints Principais

  • GET /users/ - Listar todos os usuários
  • POST /users/ - Criar novo usuário
  • POST /accounts/ - Criar nova conta bancária
  • POST /accounts/{account_id}/deposit - Realizar depósito
  • POST /accounts/{account_id}/withdraw - Realizar saque
  • POST /accounts/{account_id}/transfer - Realizar transferência

Tecnologias Utilizadas

  • FastAPI e SQLModel para o backend
  • React para interfaces de usuário
  • SQLite para persistência de dados
